#### [114. Flatten Binary Tree to Linked List](https://leetcode-cn.com/problems/flatten-binary-tree-to-linked-list/)

**Description**

Given the root of a binary tree, flatten the tree into a "linked list":

The "linked list" should use the same TreeNode class where the right child pointer points to the next node in the list and the left child pointer is always null.
The "linked list" should be in the same order as a pre-order traversal of the binary tree.

<img src="https://assets.leetcode.com/uploads/2021/01/14/flaten.jpg" alt="img" style="zoom:50%;" />

```
Input: root = [1,2,5,3,4,null,6]
Output: [1,null,2,null,3,null,4,null,5,null,6]
```



**Solution**  

* 先先序遍历一次获取遍历结果，接着从根节点开始改变左右子树，左子树置空，右子树替换成先序遍历列表中的下一个。

  

  **Code**


```cpp
class Solution {
public:
    void flatten(TreeNode* root) {
        vector<TreeNode*> order;
        preorderTraversal(root, order);
        int n = order.size();
        for (int i = 1; i < n; i++) {
            TreeNode *prev = order.at(i - 1), *curr = order.at(i);
            prev->left = nullptr;
            prev->right = curr;
        }
    }

    void preorderTraversal(TreeNode* root, vector<TreeNode*> &order) {
        if (root != NULL) {
            order.push_back(root);
            preorderTraversal(root->left, order);
            preorderTraversal(root->right, order);
        }
    }
};

```
