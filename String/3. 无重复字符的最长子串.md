#### [3. 无重复字符的最长子串](https://leetcode-cn.com/problems/longest-substring-without-repeating-characters/)

**Description**

给定一个字符串，请你找出其中不含有重复字符的 **最长子串** 的长度。

**Example**

```C++
输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
  
输入: s = ""
输出: 0

```



**Solution**

⚠️ 首先注意，不需要返回最长子串，只返回长度。

滑动窗口，当右侧的字母已经出现过时，移动左侧窗口直到它没有出现过。



**Code**

```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        int map[128] = {0};
        int len = s.size();
        int left = 0;
        int res = 0;
        for(int i=0;i<len;i++){//相当于右指针
            map[s[i]]++;
            while(map[s[i]] == 2){

                map[s[left++]]--;
            }
            res = max(res,i-left+1);
        }       
        return res;
    }
};
```
