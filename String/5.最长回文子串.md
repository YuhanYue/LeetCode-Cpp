#### [5. 最长回文子串](https://leetcode-cn.com/problems/longest-palindromic-substring/)

**Description**

给你一个字符串 `s`，找到 `s` 中最长的回文子串。

**Example**

```C++
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```



**Solution**

中心扩展

⚠️：有两种中心可以选（奇数/偶数），回文串也有奇数和偶数之分。但是这两种中心都可以用同一个函数来进行回文判断和计数处理。

将一个中心i和i+1输入到函数中，分别当作奇数中心和偶数中心，统计两种中心的回文长度，返回，比较大小，选择较大的那一个，并进行更新处理。由于返回值要求的是具体字符串，因此增加start和end用于记录字符串的起始位置。



**Code**

```cpp
class Solution {
public:
    string longestPalindrome(string s) {
        if(s.size() < 2) return s;

        int start = 0;
        int end = 0;
        int maxLen = 0;
        for(int i = 0; i < s.length(); i++){
           int len1 = helper(s,i,i);
           int len2 = helper(s,i,i+1);
           maxLen = max(max(len1, len2), maxLen);
            
            //Update
           if(maxLen > end - start + 1 ){
               start = i - (maxLen-1)/2;
               end = i + maxLen/2;
           }  
        }
         return s.substr(start,maxLen);//start from start, length is maxLen
    }

    int helper(string s, int m, int n){
        while(m>=0 && n<s.size() && s[m]==s[n]){//m>=0
            m--;
            n++;
        }
        return n-m-1;
    }
};


```
