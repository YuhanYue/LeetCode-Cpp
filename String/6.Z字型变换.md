#### [6. Z 字形变换](https://leetcode-cn.com/problems/zigzag-conversion/)

**Description**

将一个给定字符串 s 根据给定的行数 numRows ，以从上往下、从左到右进行 Z 字形排列。

比如输入字符串为 "PAYPALISHIRING" 行数为 3 时，排列如下：



**Example**

```C++
输入：s = "PAYPALISHIRING", numRows = 4
输出："PINALSIGYAHRPI"
解释：
P     I    N
A   L S  I G
Y A   H R
P     I

```



**Solution**

模拟法，模拟numsRow行数组，每一行放入对应的字符，最后将每一行拼接起来的到结果。<img src="/Users/overainy/Library/Application Support/typora-user-images/Screen Shot 2021-04-22 at 6.44.53 PM.png" alt="Screen Shot 2021-04-22 at 6.44.53 PM" style="zoom:50%;" />

**Code**

```cpp
class Solution {
public:
    string convert(string s, int numRows) {
        if(numRows == 1) return s;

         vector<string> z(min(numRows, int(s.size())));
        bool change = false;
        int rows = 0;

        for(int i = 0; i < s.size(); i++){
            z[rows].push_back(s[i]);
            if(rows == numRows-1 || rows == 0) change = !change;
            if(change) rows++;//向下
            else rows--;
        }

        string res;
        for(int i = 0; i < z.size(); i++){
            res = res + z[i];
        }
        return res;
    }
};
```
