#### [15. 三数之和](https://leetcode-cn.com/problems/3sum/)

**Description**

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有和为 0 且不重复的三元组。

**注意：**答案中不可以包含重复的三元组。

**Example**

```C++
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
  
输入：nums = [0]
输出：[]
```



**Solution**

做过两数之和，可以将三数和转化为两数和。首先排序，选定第一个数作为target，剩下的数组用双指针找和为target的相反数的两个数。

注意排除重复三元组的操作为：跳过重复的target数字。

**Code**

```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> res;

        if(nums.size() < 3){
            return res;
        }

        int i,j,k;
        k = 0;
        i = k+1;
        j = nums.size()-1;

        sort(nums.begin(), nums.end());

        for(k = 0; k < nums.size(); k++){
            if(nums[k] > 0) return res;
            if(k>0 && nums[k] == nums[k-1]) continue;

            i = k+1;
            j=nums.size()-1;
            while(i<j){
                if(nums[i]+nums[j] == -nums[k]){
                    res.push_back(vector<int>{nums[i],nums[k], nums[j]});
                    i++;
                    j--;
                    //skip duplicated number
                    //左边界left和左边的比较，右边界和右边的比较，不然会搞乱
                    while(i<j && nums[i] == nums[i-1]) i++;
                    while(i<j && nums[j] == nums[j+1]) j--;
                }
                else if(nums[i]+nums[j] < -nums[k]) i++;
                else j--;
            }
        }
        return res;

        
    }
};


```
