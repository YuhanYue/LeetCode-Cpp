#### [面试题 10.01. 合并排序的数组](https://leetcode-cn.com/problems/sorted-merge-lcci/)

**Description**

给定两个排序后的数组 A 和 B，其中 A 的末端有足够的缓冲空间容纳 B。 编写一个方法，将 B 合并入 A 并排序。

初始化 A 和 B 的元素数量分别为 m 和 n。

**Example**

```C++
输入:
A = [1,2,3,0,0,0], m = 3
B = [2,5,6],       n = 3

输出: [1,2,2,3,5,6]
```



**Solution**

1. zz做法：先将b插入，然后sort整体
2. 倒序插入

**Code**

```cpp
class Solution {
public:
    void merge(vector<int>& A, int m, vector<int>& B, int n) {
        for(int i = 0; i < n; i++){
            A[m] = B[i];
            ++m;
        }
        sort(A.begin(), A.end());
    }
};

class Solution {
public:
    void merge(vector<int>& A, int m, vector<int>& B, int n) {
        int index1 = m - 1;
        int index2 = n - 1;
        int cur = m + n - 1;

        while(index1 >= 0 && index2 >= 0){
            if(A[index1] < B[index2]){
                A[cur] = B[index2];
                --index2;
                --cur;
            }
            else if(A[index1] >= B[index2]){
                A[cur] = A[index1];
                --index1;
                --cur;
            }
        }
        if(index2>=0){
            while(index2 >= 0){
                A[cur] = B[index2];
                --index2;
                --cur;
            }
        }
    }   
};
```
