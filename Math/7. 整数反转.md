#### [7. 整数反转](https://leetcode-cn.com/problems/reverse-integer/)

**Description**

给你一个 32 位的有符号整数 x ，返回将 x 中的数字部分反转后的结果。

如果反转后整数超过 32 位的有符号整数的范围 [−231,  231 − 1] ，就返回 0



**Example**

```C++
输入：x = 123
输出：321
  
输入：x = -123
输出：-321
  
输入：x = 120
输出：21

```



**Solution**

首先想到的是1. 取模 2. 倒置 3. 判断溢出，但问题在于倒置的过程中可能已经溢出了，因此溢出的判断要放在每一次x10的时候去做；

从当前数的倒数第二位开始判断，如果除开最后一位已经超了，则溢出；如果刚好除了最后一位外相等，则单独判断最后一位。



**Code**

```cpp
class Solution {
public:
    int reverse(int x) {
        int ans = 0;
        //reverse
        int pop;
        while(x!=0){
            pop = x % 10;
            if(ans > 214748364 || (ans==214748364 && pop >7)) return 0;
            if(ans < -214748364 || (ans == -214748364 && pop < -8)) return 0;
            ans = ans*10 + pop;
            x /= 10;
        }

        return ans;
    }
};
```
