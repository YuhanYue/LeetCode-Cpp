#### [22. Generate Parentheses](https://leetcode-cn.com/problems/generate-parentheses/)

**Description**

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.



**Example**

```C++
Example 1:

Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
Example 2:

Input: n = 1
Output: ["()"]


```



**Solution**

回溯：加入左括号后才可以加入右括号

解题思路
左括号数量与 n 相同
右括号数量不大于左括号
实现思路：
left 为可添加的左括号数量，即需求数量减去当前已加入的左括号数量
right 为可添加的右括号数量，即存在左括号数量减去已加入的右括号数量

当加入左括号时 left - 1，right + 1
当加入右括号时 left 不变，right - 1
当 left 和 right 都为 0 时，即括号生成完毕后，存入数组中

**Code**

```cpp
class Solution {
public:
    vector<string> generateParenthesis(int n) {
        vector<string> strs;
        backtrace(strs,"",n,0);
        return strs;
    }

    void backtrace(vector<string>& strs,string str,int left,int right) {
        if (left == 0 && right == 0) {
            strs.push_back(str);
            return;
        }
        
        if (left > 0) backtrace(strs,str+"(",left-1,right+1);

        if (right > 0) backtrace(strs,str+")",left,right-1);
    }

};


```
