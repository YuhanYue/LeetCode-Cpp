#### [[面试题 01.08. 零矩阵](https://leetcode-cn.com/problems/zero-matrix-lcci/)](https://leetcode-cn.com/problems/rotate-matrix-lcci/)

**Description**

编写一种算法，若M × N矩阵中某个元素为0，则将其所在的行与列清零。

**Example**

```C++
输入：
[
  [1,1,1],
  [1,0,1],
  [1,1,1]
]
输出：
[
  [1,0,1],
  [0,0,0],
  [1,0,1]
]
```



**Solution**

先保存要清零的行号/列号，再对矩阵进行操作。

**Code**

```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int row = matrix.size();
        int col = matrix[0].size();

        set<int> s1;
	    set<int> s2;
        //vector<vector<bool>>& visited(row, vector<int>(col,false));

        for(int i = 0; i < row; i++){
            for(int j = 0; j < col; j++){
                    if(matrix[i][j] == 0){
                       s1.insert(i);
                       s2.insert(j);
                    }
            }
        }

        for(auto rows: s1){
            for(int i = 0; i < col; i++){
                matrix[rows][i] = 0;
            }
        }
        for(auto cols: s2){
            for(int j = 0; j < row; j++){
                matrix[j][cols] = 0;
            }
        }
    }
};
```
