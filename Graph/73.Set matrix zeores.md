#### [73. Set Matrix Zeroes](https://leetcode-cn.com/problems/set-matrix-zeroes/)

**Description**

Given an m x n integer matrix matrix, if an element is 0, set its entire row and column to 0's, and return the matrix.

You must do it in place.

一模一样：面试题01.08 0矩阵

**Example**

![img](https://assets.leetcode.com/uploads/2020/08/17/mat1.jpg)

```C++

Example 1:
Input: matrix = [[1,1,1],[1,0,1],[1,1,1]]
Output: [[1,0,1],[0,0,0],[1,0,1]]
Example 2:


Input: matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
Output: [[0,0,0,0],[0,4,5,0],[0,3,1,0]]

```



**Solution**

使用标记数组，记录要清零的元素位置。再遍历一遍数组，根据标记数组来set zero。

⚠️：判定条件要注意，不要把刚设置为true的元素又因为不等于0而设置成了false。

**Code**

```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int rows = matrix.size();
        int cols = matrix[0].size();
        vector<vector<bool>> labled (rows, vector<bool> (cols));

        for(int i = 0; i < rows; i++){
            for(int j = 0; j < cols; j++){
                if(matrix[i][j] == 0){
                    //update rows and cols
                    for(int t = 0; t < rows; t++){
                        labled[t][j] = true;
                    }
                    for(int t = 0; t < cols; t++){
                        labled[i][t] = true;
                    }
                }
                if(labled[i][j] == true) continue;
                labled[i][j] = false;
                
            }
        }

        for(int i = 0; i < rows; i++){
            for(int j = 0; j < cols; j++){
                if(labled[i][j] == true) matrix[i][j] = 0;
             }
         }

        }
};

```
