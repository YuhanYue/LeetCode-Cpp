#### [剑指 Offer 52. 两个链表的第一个公共节点](https://leetcode-cn.com/problems/liang-ge-lian-biao-de-di-yi-ge-gong-gong-jie-dian-lcof/)

**Description**

实现一种算法，找出单向链表中倒数第 k 个节点。返回该节点的值。



**Example**

```C++
输入： 1->2->3->4->5 和 k = 2
输出： 4
```



**Solution**

这是哪个小聪明蛋想出来的双指针！！！

这题要求链表的倒数第k个节点，最简单的方式就是使用两个指针：

第一个指针先移动k步，然后第二个指针再从头开始，这个时候这两个指针同时移动，当第一个指针到链表的末尾的时候，返回第二个指针即可。

<img src="https://pic.leetcode-cn.com/822a27c941efca184afcaa299c9fea3f543c73e8a20ce48529c8b5201c636525-image.png" alt="image.png" style="zoom:33%;" />

**Code**

```cpp
class Solution {
public:
    int kthToLast(ListNode* head, int k) {
        ListNode* p = head;
        ListNode* q = head;
        while(--k > 0){
            q = q->next;
        }

        while(q->next){
            p = p->next;
            q = q->next;
        }
        return p->val;
    }
};
```
