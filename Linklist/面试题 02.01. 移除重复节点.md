#### [[面试题 02.01. 移除重复节点](https://leetcode-cn.com/problems/remove-duplicate-node-lcci/)](https://leetcode-cn.com/problems/liang-ge-lian-biao-de-di-yi-ge-gong-gong-jie-dian-lcof/)

**Description**

编写代码，移除未排序链表中的重复节点。保留最开始出现的节点。

**Example**

```
 输入：[1, 2, 3, 3, 2, 1]
 输出：[1, 2, 3]
```

**Solution**

`unordered_set.count()`

**Code**

```cpp
class Solution {
public:
    ListNode* removeDuplicateNodes(ListNode* head) {
        unordered_set<int> hash;
        if(head == NULL) return NULL;

        ListNode* pre = head;
        hash.insert(head->val);

        while(pre->next){
            ListNode* cur = pre->next;
            if(!hash.count(cur->val)){
                hash.insert(cur->val);
                pre = pre->next;
            } else{
                pre->next = cur->next;
            }
        }

        return head;
    }
};
```
